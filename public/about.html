How It Works

This site utilizes RapidAPI's NBA player data and a Supabase database to fetch and store NBA player career averages. ChartJS allows for visual comparisons of the players.

01

Check SupaBase Database

Instantly queries the internal database prior to sending a request to RapidAPI

02

Fetch Player

If the player is not in the database, a request is sent to RapidAPI for the NBA player's data

03

Push to Database

The data is pushed into the database, where we pull the data from and display on the screen

The Front-End
